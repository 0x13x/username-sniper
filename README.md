## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/g3y/username-sniper.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Go the `config.json` then put your USER token in `token`
4  Start the sniper 
   ```sh
   node app.js
   ```
